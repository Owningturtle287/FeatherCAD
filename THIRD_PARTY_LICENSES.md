# Third-party software

FeatherCAD loads the following pinned ES modules from jsDelivr at runtime. They are cached by the service worker after the first successful load.

| Package | Version | Use | License |
|---|---:|---|---|
| [three](https://github.com/mrdoob/three.js) | 0.170.0 | WebGL renderer, controls, mesh loaders/exporters | MIT |

No proprietary service, API key, telemetry package, font, icon set, or cloud account is required. FeatherCAD's logo, interface icons, application code, sketch solver, and polygonal CSG implementation are original project assets released under the repository MIT license.
